# TP Réseau de Neurones XOR avec NumPy

## Description

Ce projet présente l'implémentation d'un mini réseau de neurones multicouche en Python en utilisant uniquement **NumPy** et **Matplotlib**. L'objectif est de résoudre le problème classique **XOR**, qui ne peut pas être résolu par un simple perceptron.

Le notebook contient toutes les étapes de réalisation :
- Étude des fonctions d'activation.
- Construction du réseau de neurones.
- Entraînement du modèle.
- Évaluation des performances.
- Analyse des résultats.

---

## Technologies utilisées

- Python 3
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Résultats obtenus

Le modèle obtient les performances suivantes :

- Accuracy : **95 %**
- Precision : **97.14 %**
- Recall : **91.89 %**
- F1-score : **94.44 %**
- AUC : **0.934**
